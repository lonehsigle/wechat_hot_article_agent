# 产品更新日志

## 版本: 1.6.0
## 更新日期: 2026-04-18

### 修复 Bug
1. **API响应格式统一**: 修复所有API接口响应格式不一致的问题，统一使用success字段标识成功/失败
2. **错误处理加固**: 修复多个功能在异常情况下程序崩溃的问题（热点雷达、监控、微信采集、风格管理等）
3. **参数验证**: 修复创作工坊、发布、风格管理等功能缺少参数验证导致异常的问题
4. **数值安全**: 修复多处数字参数解析异常导致数据库查询失败的问题
5. **类型安全**: 修复多处不安全的类型使用导致潜在运行时错误的问题
6. **前端数据加载**: 修复多个页面数据加载后无法正确显示的问题
7. **内存泄漏修复**: 修复多个页面切换时内存泄漏导致性能下降的问题
8. **AI功能超时**: 修复AI生成和流式输出功能无超时控制导致长时间等待的问题
9. **网页抓取超时**: 修复网页内容抓取功能无超时控制的问题
10. **AI配置一致性**: 修复热点雷达功能忽略用户AI配置始终使用DeepSeek的问题

### 性能优化
1. **LLM请求超时**: AI调用添加120秒超时控制，流式输出添加300秒超时控制
2. **爬虫超时**: 网页抓取添加30秒超时控制
3. **前端请求取消**: 页面切换时自动取消未完成的请求，减少不必要的网络流量
